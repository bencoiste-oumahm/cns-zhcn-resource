# cns-zhcn-resource

一个用于归档和发布独立 HTML 页面的资源仓库。

## 项目简介

本仓库旨在收集、整理并发布若干独立的 HTML 页面资源。这些页面内容各自独立，不依附于任何特定网站或域名。仓库本身仅作为一个静态 HTML 页面的托管与分发点，方便用户直接通过浏览器访问或引用。

## 目录说明

```
cns-zhcn-resource/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面所需的公共资源（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件代表一个完整的独立页面，可直接通过浏览器打开。
- `assets/`：存放页面共享的样式、脚本或媒体文件，便于维护和复用。

## 页面归档说明

当前仓库中的 HTML 页面均为独立归档，彼此之间没有依赖关系。每个页面都包含完整的结构与样式，可以直接在本地或通过 GitHub Pages 等服务访问。

如需新增页面，请将 HTML 文件放入 `pages/` 目录，并确保其引用的资源路径正确（建议使用相对路径）。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来补充或修正页面内容。
- 请保持页面独立性：每个 HTML 文件应自包含，避免对外部资源的硬编码依赖。
- 资源文件请统一放入 `assets/` 目录，并按类型或用途组织子文件夹。
- 本仓库不承诺更新频率，仅作为静态资源存档使用。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/cns-zhcn-resource.git
   ```
2. 直接在浏览器中打开 `pages/` 下的任意 HTML 文件即可查看。

或通过 GitHub Pages 访问（需在仓库设置中开启 Pages 功能，并选择 `main` 分支的 `/` 或 `/docs` 目录）。
